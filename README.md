# 📊 Dashboard Interativo com React

## 📖 Descrição
Dashboard desenvolvido em **React** para visualização de métricas e dados em tempo real.  
Interface responsiva e moderna, com gráficos dinâmicos.

## 🚀 Tecnologias
- React.js
- JavaScript / HTML / CSS
- Chart.js / Recharts
- Git/GitHub

## ▶️ Como Executar
1. Clone o repositório
2. Instale dependências: `npm install`
3. Execute: `npm run dev`
